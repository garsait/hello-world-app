# Hello World Web App 🌍

Una aplicación web sencilla para aprender los conceptos básicos de desarrollo web.

## Características

- ✨ Interfaz limpia y moderna
- 🎨 Diseño responsivo
- 🔄 Mensajes interactivos
- 🎯 Fácil de entender y modificar

## Cómo usar

1. Abre el archivo `index.html` en tu navegador
2. Haz clic en el botón "Cambiar Mensaje" para ver diferentes mensajes
3. Observa la animación suave al cambiar de mensaje

## Estructura del proyecto

```
hello-world-app/
├── index.html   # Estructura HTML
├── style.css    # Estilos CSS
├── script.js    # Lógica JavaScript
└── README.md    # Este archivo
```

## Tecnologías utilizadas

- **HTML5**: Estructura de la página
- **CSS3**: Estilos y animaciones
- **JavaScript (ES6)**: Interactividad

## ¿Cómo funciona?

### HTML
Define la estructura básica de la página con:
- Un contenedor principal
- Un título de bienvenida
- Un párrafo para mostrar los mensajes
- Un botón para cambiar mensajes

### CSS
Aplica estilos visuales incluyendo:
- Gradiente de fondo morado
- Tarjeta blanca centrada
- Animaciones suaves al cambiar mensajes
- Efectos hover en el botón

### JavaScript
Agrega interactividad mediante:
- Array de 6 mensajes en inglés y español
- Evento click en el botón
- Cambio dinámico de texto
- Animación fade-in al cambiar mensaje

## Mensajes incluidos

2. ¡Hola Mundo!
3. Welcome to Web Development
4. ¡Bienvenido al Desarrollo Web!
